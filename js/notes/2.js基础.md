# 日志-调试

- 输出日志不但可以判断代码是否执行，还可以判断值是否正确

```javascript
console.log("qqqq"); //日志
console.dir("bbb"); //打印对象结构
console.error("bbb"); //打印对象结构
console.trace("bbb"); //信息
console.info("bbb"); //信息

a = confirm("你是否要约会？"); //true||false

b = prompt("请输入你的年龄", "27 岁");
```
